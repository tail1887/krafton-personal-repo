# Week2 템플릿 폴더

알고리즘 문제를 `CON -> DEF -> CS -> LOG -> PRF -> TC/SC -> IMP` 흐름으로 연습하기 위한 템플릿 모음입니다.

## 사용 방법

1. `algorithm_pipeline_template.py`를 복사해 새 문제 파일을 만듭니다.
2. 코드 작성 전, 주석 섹션을 먼저 채웁니다.
3. `OUT_*`이 다음 섹션 `IN`으로 이어지는지 확인합니다.
4. 마지막에 `IMP`를 구현하고 edge case를 먼저 테스트합니다.

## 약어

- `CON`: 제약/입력 분석
- `DEF`: 상태/관계/목표 정의
- `CS`: 후보 접근 비교, 경우의 수, 전략 선택
- `LOG`: 의사코드/처리 순서
- `PRF`: 정당성 근거
- `TC/SC`: 시간/공간 복잡도
- `IMP`: 구현/검증
